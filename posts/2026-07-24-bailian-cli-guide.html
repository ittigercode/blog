🎨 阿里云百炼 CLI 安装与 AI 画图实战

2026-07-24 · 从零到生成第一张 AI 图片
阿里云百炼 CLI（bailian-cli）是阿里云官方命令行工具，支持文本对话、图片生成、应用管理等。本文从安装到鉴权到实战，10 分钟搞定。

一、环境要求

依赖最低版本检查命令
Node.js≥ 22.12.0node -v
npm任意新版npm -v

二、安装 CLI

npm install -g bailian-cli
验证： 
bl --version   # 应输出版本号
which bl       # 确认在 PATH 中
注意：只能用 npm 安装，不能用 pnpm 或 yarn。如果报 engines 错误，升级 Node.js。

三、安装 Skills

npx skills add modelstudioai/cli --all -g
这会把百炼的 Agent Skill 安装到所有支持的 AI Agent 工具中（Cursor、Claude Code、Hermes 等）。

四、鉴权（二选一）

方式 A：浏览器登录（推荐）

bl auth login --console
自动拉起浏览器完成阿里云控制台授权，不用手动复制 Key。

方式 B：API Key 登录

  1. 去 百炼控制台 获取 API Key
  2. bl auth login --api-key ***
验证： 
bl auth status --output json

五、实战：AI 画图

5.1 一句话生成图片

bl image generate --prompt "纯黑色夏日男装T恤产品主图"

5.2 支持的能力

| 功能 | 命令 | |------|------| | 💬 文本对话 | `bl text chat --message "你好"` | | 🎨 图片生成 | `bl image generate --prompt "..."` | | 📱 应用列表 | `bl app list` | | 📊 用量查询 | `bl usage free` |

5.3 图片输出路径

生成的图片默认保存到 ~/bailian-output/images/。

六、与 Hermes 配合

百炼 CLI 的 Key 和 DashScope 是同一套，配置到 Hermes：

# ~/.hermes/.env
DASHSCOPE_API_KEY=***   # 百炼 Key 直接复用
DASHSCOPE_WORKSPACE_ID=你的ID
DASHSCOPE_REGION=cn-beijing

配好后，手绘幻灯片 Skill 就能用这个 Key 生成图片了。

七、总结

步骤耗时
npm install10 秒
npx skills add20 秒
bl auth login10 秒
生成第一张图5 秒

全过程不到 1 分钟。百炼 CLI 弥合了 Hermes 不能直接画图的短板，Key 复用 DashScope，一套凭据两处用。
